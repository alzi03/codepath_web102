# Web Development Project 6 - Kingdom Hearts Characters

Submitted by: Alam Zishan

This web app: Displays a list of Kingdom Hearts characters coupled with a list of films, video games, and tv shows each appears in. When viewing each individual character, the user can view another category park attractions. The website also features two charts on the home page that display the total number of films for all the charavcters, as well as the most popular TV shows. Each individual character features a chart that shows the number of films, tv shows, park attractions, and video games they appear in.
